# 🏥 Hospital Management System (SQL)

This project is a simple **Hospital Management Database** built using SQL. It manages patients, doctors, appointments, admissions, billing, and more.

## 📌 Features

* Patient & Doctor management
* Appointment scheduling
* Admission & Discharge tracking
* Billing system
* Automated triggers for billing and status updates

## 🗂️ Tables Included

* Patients
* Doctors
* Nurses
* StaffRoster
* Appointments
* Admission
* Discharge
* Billing

## ⚙️ Technologies Used

* MySQL / SQL

## 🚀 How to Use

1. Run the SQL script in MySQL
2. Database and tables will be created automatically
3. Sample data will be inserted
4. Use SELECT queries to view records

## 🔁 Automation

* Trigger for billing after appointment completion
* Trigger for updating admission status after discharge  
* Trigger for automatic billing after discharge
